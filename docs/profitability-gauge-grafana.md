# Gauge-метрика доходности в Grafana

Документ описывает панель Gauge в Grafana, которая показывает доходность в рублях и меняет цвет по порогам:

- до `100 000` руб. - красный;
- от `100 000` до `200 000` руб. - желтый;
- от `200 000` руб. и выше - зеленый.

## Доступ к Dashboard

Grafana доступна по адресу:

```text
http://localhost:3001
```

Логин и пароль:

```text
admin / admin
```

Готовый dashboard:

```text
http://localhost:3001/d/profitability-gauge-demo/dohodnost
```

## Источник данных

Используется таблица `payments` из основной базы PostgreSQL.

SQL-запрос для метрики (доходность за 30 дней):

```sql
SELECT COALESCE(SUM(amount), 0)::double precision AS "Доходность"
FROM payments
WHERE status = 'completed'
  AND payment_date >= NOW() - INTERVAL '30 days';
```

Доступные периоды:

- 7 дней
- 30 дней
- 90 дней
- Все время

## Dashboard

Dashboard содержит 7 панелей Gauge:

**Реальные данные (из payments):**
1. Доходность за 7 дней (пороги: 50 000 / 100 000 руб.)
2. Доходность за 30 дней (пороги: 50 000 / 100 000 руб.)
3. Доходность за 90 дней (пороги: 100 000 / 200 000 руб.)
4. Доходность (все время) (пороги: 100 000 / 200 000 руб.)

**Тестовые примеры:**
5. Пример: 50 000 руб. (красная зона)
6. Пример: 150 000 руб. (желтая зона)
7. Пример: 250 000 руб. (зеленая зона)

## Настройка Thresholds (тестовые панели)

| Value | Color | Значение |
| --- | --- | --- |
| `null` | red | до `100 000` руб. |
| `100000` | yellow | от `100 000` до `200 000` руб. |
| `200000` | green | от `200 000` руб. и выше |

В JSON dashboard эта настройка выглядит так:

```json
{
  "mode": "absolute",
  "steps": [
    { "color": "red", "value": null },
    { "color": "yellow", "value": 100000 },
    { "color": "green", "value": 200000 }
  ]
}
```
