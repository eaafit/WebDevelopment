# Тексты шаблонов электронной почты

Шаблоны для EmailJS (или другого почтового провайдера). Переменные в фигурных скобках подставляются из кода (`to_email`, `reset_url`, `app_name` и т.д.).

---

## 1. Восстановление пароля

**Тема:** `{{app_name}} — восстановление пароля`

**Текст:**

```
Здравствуйте.

Вы запросили восстановление пароля в {{app_name}}.

Перейдите по ссылке, чтобы задать новый пароль:
{{reset_url}}

Ссылка действует ограниченное время. Если вы не запрашивали сброс, проигнорируйте это письмо.

С уважением,
команда {{app_name}}
```

**Переменные:** `to_email`, `reset_url`, `app_name`

---

## 2. Регистрация / создание учётной записи

**Тема:** `Добро пожаловать в {{app_name}}`

**Текст:**

```
Здравствуйте, {{full_name}}!

Ваша учётная запись в {{app_name}} успешно создана.

Роль: {{role_label}}
Вход в личный кабинет: {{login_url}}

Если вы не регистрировались в сервисе, свяжитесь с поддержкой.

С уважением,
команда {{app_name}}
```

**Переменные:** `to_email`, `full_name`, `role_label`, `app_name`, `login_url`

Для сценария «администратор создал пользователя» можно заменить первый абзац на: «Администратор создал для вас учётную запись…» и при необходимости вместо `login_url` использовать `set_password_url`.

---

## 3. Заявитель отправил новую заявку (подтверждение заявителю)

**Тема:** `{{app_name}} — заявка принята`

**Текст:**

```
Здравствуйте, {{full_name}}!

Мы получили вашу заявку.

Кратко о запросе:
{{request_summary}}

Дата и время отправки: {{submitted_at}}

Отслеживать статус можно в разделе «Мои заявки»: {{orders_url}}

С уважением,
команда {{app_name}}
```

**Переменные:** `to_email`, `full_name`, `request_summary`, `submitted_at`, `app_name`, `orders_url`

---

## 4. Новая заявка (уведомление сотруднику / нотариусу)

**Тема:** `[{{app_name}}] Новая заявка от заявителя`

**Текст:**

```
Поступила новая заявка.

Заявитель: {{full_name}} ({{applicant_email}})

Содержание:
{{request_summary}}

Время: {{submitted_at}}

Откройте панель администратора или назначенный раздел портала для обработки.
```

**Переменные:** `to_email` (получатель), `applicant_email`, `full_name`, `request_summary`, `submitted_at`, `app_name` (в теме)

---

## HTML-версия

В EmailJS тело можно оформить теми же абзацами через простые теги `<p>`, ссылки — `<a href="{{login_url}}">` и т.п.
