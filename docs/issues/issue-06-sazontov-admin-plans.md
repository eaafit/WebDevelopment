## Описание

Раздел панели администратора для управления тарифными планами подписок, скидками и промокодами.

## Затронутые роли

- Администратор

## Экраны / компоненты

- [ ] **Список тарифных планов** (`/admin/plans`) — таблица: название, цена, период, статус активности; кнопки создать/редактировать/деактивировать
- [ ] **Форма тарифного плана** — создание и редактирование: название, описание, цена, валюта, период подписки, флаг активности
- [ ] **Список скидок** — таблица скидок, привязанных к тарифам
- [ ] **Список промокодов** (`/admin/plans` или отдельный маршрут) — таблица: код, скидка (%), лимит использований, использовано, дата истечения
- [ ] **Форма промокода** — создание/редактирование промокода: код, процент скидки, лимит, дата истечения

## Технические требования

- Маршруты: `/admin/plans`, `/admin/plans/:id/edit`
- Защита: `roleGuard(UserRole.Admin)` из `@notary-portal/ui`
- RPC: `PaymentService` (методы `GetSubscription`, `CreateSubscription`); промокоды — согласовать с бэкендом
- Модальные окна подтверждения для деактивации плана / удаления промокода

## Acceptance criteria

- [ ] Видна таблица всех тарифных планов с актуальными ценами
- [ ] Можно создать, отредактировать и деактивировать тарифный план
- [ ] Можно создать промокод с ограничением на количество использований и сроком действия
- [ ] Система отображает, сколько раз использован каждый промокод

## Связанные файлы

- `libs/web/admin/src/lib/features/plans/`
- Proto-контракты: `PaymentService`
- Модели Prisma: `Subscription`, `Promo`
