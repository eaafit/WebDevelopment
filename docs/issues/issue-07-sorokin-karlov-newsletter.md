## Описание

Раздел панели администратора для просмотра списка рассылки и отправки email-рассылок пользователям системы.

> **Разработчики:** Сорокин Д.Э. и Карлов И.А. (DaEtoJostka) — реализуют совместно.  
> Карлов И.А. отвечает за функциональную часть данного issue.

## Затронутые роли

- Администратор

## Экраны / компоненты

- [x] **Список рассылки** (`/admin/newsletter`) — таблица: email, ФИО, дата подписки, статус подписки (активна/отписан); фильтр, поиск, пагинация
- [x] **Форма создания рассылки** — выбор аудитории (все / по роли / выбранные), тема письма, HTML/текстовый редактор тела письма, кнопка «Предпросмотр» и «Отправить»
- [x] **Страница истории рассылок** — список отправленных кампаний: дата, тема, кол-во получателей, статус доставки

## Технические требования

- Маршруты: `/admin/newsletter`, `/admin/newsletter/new`
- Защита: `roleGuard(UserRole.Admin)` из `@notary-portal/ui`
- Редактор письма: встроенный `<textarea>`
- RPC: `notary.newsletter.v1alpha1.NewsletterService`
- Подтверждение перед массовой отправкой (модальное окно)

## Acceptance criteria

- [x] Администратор видит полный список подписчиков с возможностью поиска
- [x] Можно выбрать аудиторию и составить письмо
- [x] Перед отправкой доступен предпросмотр
- [x] После отправки отображается подтверждение и запись в истории
- [x] История показывает статус доставки каждой кампании
