## Описание

Реализация справочного раздела: база знаний / FAQ со статьями, структурированной навигацией по категориям.

> **Разработчики:** Титов Святослав (TandZir0) — структура базы знаний и статьи.  
> Рахманов Рахман (LaironAS) — поиск по статьям (issue-13).

## Затронутые роли

- Нотариус
- Заявитель (если оплатил доступ)

## Экраны / компоненты

- [ ] **Главная справочника** (`/faq`) — список категорий / разделов, рекомендованные статьи, поиск (делегируется issue-13)
- [ ] **Страница категории** (`/faq/category/:slug`) — список статей категории с краткими анонсами
- [ ] **Страница статьи** (`/faq/article/:slug`) — полный текст статьи, навигация «предыдущая / следующая», хлебные крошки
- [ ] **Блок «Популярные статьи»** — на главной справочника, топ-5 по просмотрам

## Технические требования

- Маршруты: `/faq`, `/faq/category/:slug`, `/faq/article/:slug`
- Защита: аутентифицированный пользователь (или тарифная проверка для заявителей)
- Контент статей: HTML / Markdown, рендерить безопасно (`DOMPurify` или Angular `innerHtml` pipe)
- RPC: согласовать endpoint для статей с бэкендом (или использовать статичные JSON-файлы на первом этапе)

## Acceptance criteria

- [ ] Пользователь видит список категорий и может перейти в любую
- [ ] Внутри категории отображается список статей
- [ ] Статья отображается с форматированием (заголовки, списки, изображения)
- [ ] Хлебные крошки показывают путь навигации
- [ ] Работает интеграция с поиском (issue-13 Рахманов Рахман)
