## Как создать issues в GitHub

```bash
# С помощью GitHub CLI (gh):
for file in docs/issues/issue-*.md; do
  title=$(grep '^title:' "$file" | sed 's/title: //' | tr -d '"')
  labels=$(grep '^labels:' "$file" | sed 's/labels: //' | tr -d '[]"' | tr ',' '\n' | xargs -I{} echo -n '{}, ' | sed 's/, $//')
  body=$(tail -n +6 "$file")
  gh issue create --title "$title" --body "$body" --label "$labels"
done
```

Или создать вручную через GitHub UI, скопировав содержимое каждого файла.
