## Описание

Переиспользуемый компонент загрузки и управления файлами (PDF/изображения): drag&drop, валидация, предпросмотр, переименование, удаление, версионирование, теги, статусы модерации.

> **Этот компонент является shared — используется в заявках, форме наследства, чате поддержки и копиях документов.**  
> Разместить в `libs/web/shared/ui/src/lib/file-upload/`.

## Затронутые роли

- Заявитель / Нотариус / Администратор (по правам)

## Экраны / компоненты

- [ ] **Зона Drag&Drop** — визуальный дроп-таргет, подсветка при перетаскивании, fallback-кнопка выбора файла
- [ ] **Список загруженных файлов** — имя, размер, тип, статус («принято» / «на проверке» / «отклонено»), иконка типа
- [ ] **Предпросмотр** — изображения: миниатюра inline; PDF: открытие в modal через `<iframe>` или `pdf.js`
- [ ] **Переименование** — inline-редактирование имени файла (double click / иконка)
- [ ] **Удаление** — с подтверждением
- [ ] **Теги** — добавление меток (фасад / план / право / другое), мультивыбор
- [ ] **Версии** — список предыдущих версий файла, восстановление версии
- [ ] **Статус модерации** — badge «на проверке» / «принято» / «отклонено», причина при отклонении

## Технические требования

- Компонент: `FileUploadComponent` в `libs/web/shared/ui/src/lib/file-upload/`
- Экспортировать из `@notary-portal/ui`
- Валидация: форматы (PDF, JPG, PNG, DOCX), максимальный размер файла (настраивается через `@Input`)
- RPC: `DocumentService.Create`, `DocumentService.Delete`, `DocumentService.List`
- `@Input() accept`, `@Input() maxSizeMb`, `@Input() multiple`, `@Input() allowedTags`
- `@Output() filesChange: EventEmitter<UploadedFile[]>`

## Acceptance criteria

- [ ] Файлы принимаются через drag&drop и через диалог выбора
- [ ] Неподходящие форматы/размеры отклоняются с понятным сообщением
- [ ] Изображения показывают миниатюру, PDF — иконку с кнопкой предпросмотра
- [ ] Файл можно переименовать, удалить, добавить/изменить тег
- [ ] Статус модерации отображается как badge
- [ ] Компонент работает в составе форм заявок, чата поддержки и копий документов
