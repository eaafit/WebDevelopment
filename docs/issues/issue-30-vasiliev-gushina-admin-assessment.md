## Описание

Административная панель модуля оценки: очередь запросов на обработку, ручная модерация, метрики конверсии и времени, отчёты и выгрузки.

> **Разработчики:** Васильев Степан и Гущина Мария (mgusssh) — работают совместно над данным разделом.

## Затронутые роли

- Администратор

## Экраны / компоненты

- [ ] **Очередь оценок** (`/admin/assessment`) — таблица запросов в статусе «В очереди» / «В работе»: объект, заявитель, нотариус, дата, приоритет; кнопка «Назначить нотариуса»
- [ ] **Ручная модерация** — возможность вручную изменить статус оценки, добавить/заменить нотариуса, указать причину ручного вмешательства
- [ ] **Дашборд метрик** (`/admin/assessment/analytics`) — ключевые показатели:
  - Конверсия (создано → завершено), в %
  - Среднее время обработки заявки
  - Распределение по статусам (pie/donut chart)
  - Динамика заявок по периодам (line chart)
- [ ] **Отчёты и выгрузки** — кнопки «Выгрузить CSV» по периоду, параметрам фильтра

## Технические требования

- Маршруты: `/admin/assessment`, `/admin/assessment/analytics`
- Защита: `roleGuard(UserRole.Admin)` из `@notary-portal/ui`
- RPC: `AssessmentService` (все методы: `List`, `Update`, `Verify`, `Complete`, `Cancel`)
- Графики: `recharts` или `Chart.js`
- Экспорт CSV: по текущим фильтрам

## Acceptance criteria

- [ ] Очередь оценок отображается с актуальными статусами
- [ ] Администратор может назначить нотариуса или вручную изменить статус
- [ ] Метрики отображаются с актуальными данными и корректными графиками
- [ ] Выгрузка CSV работает для любого диапазона дат
