## Описание

Страница итоговых результатов оценки недвижимости: итоговая стоимость, диапазон и уверенность модели, сравнительные данные, PDF-отчёт, скачивание, комментарии.

## Затронутые роли

- Заявитель
- Нотариус

## Экраны / компоненты

- [ ] **Карточка результата** (`/applicant/assessment/results/:id`) — итоговая оценочная стоимость (крупно), диапазон (мин–макс), индикатор уверенности модели (%)
- [ ] **Блок «Сравнения»** — таблица аналогов: адрес, площадь, цена за кв.м, корректировка; визуализация позиции объекта относительно аналогов
- [ ] **Секция «Отчёт»** — кнопка «Скачать PDF», статус подписи (Draft / Signed), дата подписания
- [ ] **Комментарии нотариуса/оценщика** — текстовый блок с пояснением к результату
- [ ] **Кнопка «Запросить пересмотр»** — если заявитель не согласен с результатом (согласовать с бэкендом)

## Технические требования

- Маршруты: `/applicant/assessment/results/:id`, `/notary/assessment/results/:id`
- Защита по роли
- RPC: `AssessmentService.Get`, `ReportService.List`, `ReportService.Sign`
- Диаграмма сравнений: `recharts` или `Chart.js` (горизонтальная bar chart аналогов)
- PDF-скачивание: `<a [href]="pdfUrl" download>` или получить Blob через RPC

## Acceptance criteria

- [ ] Итоговая стоимость отображается крупным шрифтом с диапазоном
- [ ] Таблица аналогов заполнена и визуально понятна
- [ ] PDF-отчёт скачивается по кнопке
- [ ] Статус подписи отображается корректно (черновик / подписан)
- [ ] Нотариус может увидеть и добавить комментарий к результату
