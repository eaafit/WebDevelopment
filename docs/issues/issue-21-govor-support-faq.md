## Описание

Расширение чата поддержки: интеграция базы знаний/FAQ внутри интерфейса поддержки, поиск по статьям прямо из окна тикета.

> **Васорин Иван (issue-11):** базовый чат, тикеты, вложения, SLA.  
> **Говор Сергей:** FAQ-интеграция в чате, поиск по статьям, работа с базой знаний из интерфейса поддержки.

## Затронутые роли

- Все роли

## Экраны / компоненты

- [ ] **Боковая панель «Похожие статьи»** в окне тикета — при создании/просмотре тикета автоматически предлагаются статьи FAQ по ключевым словам темы
- [ ] **Поиск по базе знаний** внутри чата поддержки — строка поиска, результаты в боковой панели, клик открывает статью
- [ ] **Ссылки из статей в тикет** — агент поддержки может вставить ссылку на статью в ответ одним кликом
- [ ] **Страница базы знаний для поддержки** (`/support/faq`) — агентский вид: категории + поиск + статьи

## Технические требования

- Автоматический поиск по FAQ срабатывает при создании тикета (debounce по заголовку)
- Переиспользовать поисковый компонент из issue-13 (Рахманов Рахман)
- Маршрут: `/support/faq`
- RPC / endpoint для поиска FAQ: согласовать с бэкендом

## Acceptance criteria

- [ ] При вводе темы тикета автоматически появляются рекомендованные статьи
- [ ] Поиск по базе знаний работает из окна поддержки
- [ ] Найденную статью можно открыть в панели рядом с тикетом
- [ ] Агент может вставить ссылку на статью в ответ одним действием
