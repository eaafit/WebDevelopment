# UI — страница географии объектов оценки (admin)

## Описание

Разработать страницу в админ-панели для визуализации географии объектов оценки на интерактивной карте. Страница доступна только роли Администратор и отображает объекты оценки (заявки) в виде маркеров на карте с возможностью фильтрации и перехода к карточке заявки.

## Контекст

- Требование: [Responsobility.md](../docs/Responsobility.md) — раздел «Личный кабинет администратора / География объектов оценки».
- Карта сайта: [sitemap.md](../docs/sitemap.md) — маршрут `/admin/geography`.
- Сущности: [Assessment](../docs/entities.md) (поля Address, Latitude, Longitude для отображения на карте).
- Роль: Администратор.

## Требования к карте

- **Компонент:** Leaflet — использовать библиотеку Leaflet для отображения карты.
- **Данные:** отображение объектов оценки (заявок) в виде маркеров по координатам (Latitude, Longitude) или по адресу с геокодированием при отсутствии координат.
- **Фильтры:** по статусу заявки, по периоду создания (дата от/до).
- **Взаимодействие:** при клике по маркеру — всплывающая подсказка (popup) с краткой информацией об объекте и ссылкой на карточку заявки; по клику переход к `/admin/orders` с открытием соответствующей заявки (или отдельный маршрут карточки заявки).

## UI/UX

- Адаптивная вёрстка (desktop и при необходимости планшет).
- Соответствие существующему стилю и навигации приложения.
- Доступ только для роли Администратор.
- Панель фильтров над или рядом с картой (статус, период).

## Ожидаемый результат

- Страница с картой (Leaflet), интегрированная в маршрут `/admin/geography`.
- Маркеры по объектам оценки с popup и ссылкой на заявку.
- Рабочие фильтры по статусу и периоду.
- Адаптивная вёрстка и соответствие стилю приложения.

## Дополнительно

- При отсутствии API для списка заявок с координатами использовать моки/стабы.
- Геокодирование адреса в координаты (если Latitude/Longitude не заданы) — опционально на первом этапе; можно отображать только объекты с заполненными координатами или подключить сервис геокодирования позже.
