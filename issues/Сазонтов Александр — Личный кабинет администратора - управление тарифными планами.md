## Ключевые функции

### Тарифные планы (`/admin/plans`)
- Сетка карточек планов: **Basic**, **Premium**, **Enterprise**
- Каждая карточка: название, описание, цена/месяц, список возможностей (ul)
- Значения из `SubscriptionPlan` enum + mock-расширение (цены, описания)
- Кнопка «Редактировать» → открывает `PlanModalComponent` (форма)
- Секция «Промокоды» ниже карточек:
  - Таблица: код, скидка (%), лимит использований, использовано, дата истечения, статус (активен/истёк)
  - Mock-данные: 5–8 записей в `promo-codes.mock.ts`
  - Кнопка «Добавить промокод» → модальное окно с формой
  - Кнопка «Деактивировать» в строке (mock — переключить статус)

### Активные подписки (`/admin/subscriptions`)
- Таблица: userId, план, дата начала, дата окончания, статус
- Mock-данные (реальный admin-метод не реализован)
- Кнопка «Аннулировать» (mock)

### `PlanModalComponent`
- Поля: название, описание, цена, список возможностей (textarea)
- Кнопки: «Сохранить», «Отмена»
- Режим создания и редактирования
