## Ключевые функции

### Список платежей (`/admin/payments`)
- Кнопки над таблицей:
  - **«Добавить платёж»** → `/admin/payments/new` (компонент Липовцева)
  - **«Экспорт CSV»**
- Таблица:
  - ID транзакции (`transactionId`)
  - userId
  - Тип (`PaymentType` → по-русски: «Подписка» / «Оценка» / «Копия документа»)
  - Сумма + валюта (форматировать: `1 234,56 ₽`)
  - Статус (`PaymentStatus` → цветной бейдж)
  - Дата (`paymentDate` → `dd.MM.yyyy HH:mm`)
- Фильтры:
  - Статус — мультиселект из `PaymentStatus` enum
  - Тип — select из `PaymentType` enum
  - Дата «от» / «до» — два `input[date]`
- Пагинация
- Клик по строке → открывает `PaymentDetailPanelComponent`
- Кнопка «Изменить» в строке → `/admin/payments/:id/edit` (компонент Липовцева)

### `PaymentDetailPanelComponent`
- Все поля: id, transactionId, userId, type, amount, currency, status, paymentDate, paymentMethod, subscriptionId, assessmentId, attachmentFileName
- Ссылка на вложение если есть `attachmentFileUrl`
- Кнопки: «Редактировать» → `/admin/payments/:id/edit`, «Удалить» → открывает `PaymentDeleteModal` (компонент Липовцева)

### Экспорт CSV
- Клиентский экспорт через `Blob`:
  ```typescript
  const csv = payments.map(p => [p.transactionId, p.userId, p.amount, ...].join(',')).join('\n');
  const blob = new Blob([csv], { type: 'text/csv' });
  ```

## Фасад-сервис
Создать `libs/web/admin/src/lib/api/admin-payments-api.service.ts`:
```typescript
@Injectable({ providedIn: 'root' })
export class AdminPaymentsApiService {
  private readonly client = createClient(PaymentService, inject(RPC_TRANSPORT));
  listPayments(query: PaymentQuery): Observable<PaymentHistoryPage>
}
```
> Использовать маппинг как в `TransactionsApiService` — не дублировать, а переиспользовать логику

## Технические требования
- `PaymentStatus`, `PaymentType` enum из `@notary-portal/api-contracts`
- Timestamp → строку через `timestampDate` из `@bufbuild/protobuf/wkt`
- Суммы через `Intl.NumberFormat` или вручную: `(1234.56).toLocaleString('ru-RU') + ' ₽'`
- Обновить `admin.routes.ts`
