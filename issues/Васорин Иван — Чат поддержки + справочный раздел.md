## Ключевые функции

### `SupportPageComponent` — двухколоночный layout
- **Левая колонка:** `TicketListComponent`
- **Правая колонка:** `TicketChatComponent` выбранного тикета

### `TicketListComponent`
- Список тикетов: тема, статус (бейдж), последнее сообщение, время
- SLA-индикатор: зелёный (> 2ч до дедлайна), жёлтый (< 2ч), красный (нарушен)
- Кнопка **«Новый тикет»** → открывает `TicketCreateFormComponent`
- Активный тикет — выделен

### `TicketChatComponent`
- Хронологическая лента сообщений
- Сообщения пользователя — справа, поддержки — слева
- Прикреплённые файлы: иконка + имя + кнопка скачивания
- **Форма отправки** внизу:
  - Textarea (Enter = новая строка, Ctrl+Enter = отправить)
  - Кнопка прикрепить файл (mock — просто показывает имя)
  - Кнопка отправить
- Отправка → добавить в mock `messages` оптимистично

### `TicketCreateFormComponent`
- Поля: «Тема» (text), «Описание» (textarea), «Прикрепить файл» (mock)
- Кнопка «Создать» → добавить в mock список тикетов

## SLA-цвета и статусы
- `open` — синий, `in_progress` — жёлтый, `resolved` — зелёный, `closed` — серый
- SLA нарушен (`slaBreached: true`) → красная полоска слева у карточки тикета
