Индивидуальное задание по структуре из `docs/coursework/structure.md`. Зона ответственности: `docs/Responsobility.md` (общий ЛК администратора: заказы, статусы, подписка, документы).

- `README.md`, `apps/nexus-web/Dockerfile`, `apps/nexus-web/DOCKER.md`.

## Раздел 2. Ваш компонент (≈4 стр.)

### 2.1. Диаграмма классов UML

По `docs/entities.md`: **User**, **Assessment** (управление статусами с точки зрения администратора), **Document**, при необходимости **Subscription** / **Payment** для сценариев администрирования заказов.

### 2.2. Диаграмма компонентов UML

Админ-кабинет → API → сервисы заявок и пользователей → БД и хранилище.
