## Примеры для разных типов заданий

- Используйте сущности: Assessment, RealEstateObject, Document
- API-методы начинаются с `/api/assessments/`

### Для заданий по админ-панели:
- Используйте сущности: User, Assessment, Payment, AuditLog
- API-методы начинаются с `/api/admin/`
- Ссылки на код: `libs/web/admin/src/lib/features/`

### Для заданий по платежам:
- Используйте сущности: Payment, TariffPlan, Subscription
- API-методы начинаются с `/api/payments/`
- Внешние зависимости: платежный шлюз (ЮKassa)

### Для заданий по уведомлениям/чатам:
- Используйте сущности: Notification, User
- Добавьте концептуальные классы для чатов/тикетов
- Внешние зависимости: SMTP, WebSocket
