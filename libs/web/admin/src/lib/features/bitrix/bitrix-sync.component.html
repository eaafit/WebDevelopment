Управление синхронизацией

info Синхронизация создаст или обновит контакты в Bitrix24 для всех активных пользователей системы. Принудительная пересинхронизация обновит все контакты, даже если данные не изменились.

Что происходит при синхронизации:

  • Система получает список всех активных пользователей
  • Для каждого пользователя ищется существующий контакт в Bitrix24 по телефону или email
  • Если контакт найден - он обновляется, если нет - создается новый
  • Создаются контакты с полями: ФИО, телефон, email, ID пользователя в системе
  • Все действия логируются для отслеживания
