Как получить доступ к Bitrix24 API:

  1. Войдите в ваш Bitrix24 портал
  2. Перейдите в раздел "Приложения" → "Мои приложения"
  3. Создайте новое веб-приложение или используйте существующее
  4. Скопируйте:
    • URL портала: домен вашего Bitrix24 (например: portal.bitrix24.ru)
    • Member ID: идентификатор пользователя
    • Access Token: токен доступа с правами на crm.contact.*, crm.deal.*
